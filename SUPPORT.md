# 获取支持

感谢你使用 VireFS！以下是获取帮助的渠道指引。

## 使用咨询

如果你有关于如何使用 VireFS 的问题：

1. 查阅 [README](README.md) 中的文档和示例
2. 搜索 [已有的 Issue](https://github.com/lin-snow/VireFS/issues) 看是否已有相关讨论
3. 如果未找到答案，欢迎创建一个 [Discussion](https://github.com/lin-snow/VireFS/discussions)（如已启用）或 Issue

## 报告 Bug

请使用 [Bug 报告模板](https://github.com/lin-snow/VireFS/issues/new?template=bug_report.yml) 提交，并尽量提供最小可复现代码。

## 功能建议

请使用 [功能建议模板](https://github.com/lin-snow/VireFS/issues/new?template=feature_request.yml) 描述你的使用场景和建议方案。

## 安全问题

**请不要通过公开 Issue 报告安全漏洞。** 请参阅 [SECURITY.md](SECURITY.md) 了解安全问题的报告流程。

## 贡献代码

想要为项目做出贡献？请阅读 [CONTRIBUTING.md](CONTRIBUTING.md) 了解开发流程和规范。
